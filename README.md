#ParquiDigital Api
Concepto parquímetro digital

## Stack
- Apollo Graphql
- Mongodb

## Uso

```
docker-compose build
docker-compose up -d
```

Puede acceder al proyecto en 

http://localhost:9000
http://localhost:9000/graphql

__Para la aplicación movil deberá exponer el puerto a la red local__

Ejemplo en ubuntu
```
sudo ufw allow 9000
```
